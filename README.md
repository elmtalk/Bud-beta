# Bud 0.10.20 beta

Fixes:
- PWA manifest/icon links are now embedded in index.html as well as included as separate files.
- App icon set rebuilt from Bud&Leaf logo.
- Service worker registration added to index.html.

Upload the CONTENTS of this ZIP to the GitHub repo root:
- index.html
- manifest.webmanifest
- service-worker.js
- icons/
- README.md

Then in Chrome:
- open the GitHub Pages URL
- refresh once
- menu > Add to Home screen / Install app
