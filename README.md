# Bud 0.10.21 beta - flat-root PWA package

This version is for simple GitHub web upload where all files are uploaded into the repo root.

Upload all files in this ZIP directly to the repo root.

Expected root files:
- index.html
- manifest.webmanifest
- service-worker.js
- icon-72.png
- icon-96.png
- icon-128.png
- icon-144.png
- icon-152.png
- icon-180.png
- icon-192.png
- icon-256.png
- icon-384.png
- icon-512.png
- README.md

Then open the GitHub Pages URL:
https://elmtalk.github.io/Bud-beta/

Chrome will open it as a webpage first. Use the three-dot menu and choose Install app / Add to Home screen.
